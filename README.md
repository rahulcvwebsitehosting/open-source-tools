## Overview

This directory catalogs open-source web-based tools that respect your privacy. No signups, no spam, no dark patterns. Every tool listed is free, open-source, and usable directly in your browser without creating an account.

### Site Structure

- **Root URL:** `/` - Main page with search, category filters, and tool grid
- **Modals (JavaScript-triggered, no separate URLs):**
  - **Submit a Tool** - Form to submit a new tool for inclusion
  - **Report a Tool** - Form to report an issue with an existing tool
  - **Suggest a Tool** - Form to suggest a new tool to be built

## How to Search

This is a static Markdown directory. To find specific tools quickly:

- **On GitHub:** Press `t` to activate the file finder, or use `Ctrl+K` / `Cmd+K` for the command palette
- **In your browser:** Use `Ctrl+F` / `Cmd+F` to search within this page
- **Browse by category:** Click a category name in the table below to see related tools
